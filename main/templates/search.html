Indian Premier League Matches Search Box

How to Search:

  • To find all items in a class, type [class_name] (available class names: "cities", "matches", "players", "seasons", "teams", "umpires", "venues").
  • e.g. type [cities] to find all cities recorded in the dataset.
  • To find items from all classes whose identifier contains a substring, type [substring].
  • e.g. type [stadium] to find all items whose name contain the substring "stadium".
